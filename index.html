Needham–Schroeder protocol

Цей програмний додаток відтворює роботу протоколу автентифікації Нідгема — Шредера.

Інструкції для роботи з програмним додатком:

  • Згенеруйте довірені ключі для Alice та Bob за допомогою кнопок в Key distribution center.
  • Згенеруйте псевдовипадкові числа для Alice та Bob
  • Для встановлення з'єднання введіть в поле повідомлення для Alice Ім'я сторони яка встановлює з'єднання, Ім'я сторони з ким потрібно встановити з'єднання, та знегероване число для Alice. Приклад: alice,bob,1234
  • Якщо все було введено вірно та автентифікація пройшла успішно ви побачите повідомлення про те що з'єднання з іншою було успішно встановлено. Після цього можна починати обмін даними.
